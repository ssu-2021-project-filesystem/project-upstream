## 코딩
> ### 1. coding convention
> 1. if, while, for 등에서 {}는 다음 행부터 작성하기.
> 2. if, while, for 문 등에서는 항상 복합문 사용하기.
> 3. 매크로 작성 시 괄호로 묶기.
> 4. 매크로, 사용자 정의형 등의 이름은 전부 대문자로 작성하기.
> 5. 연산자 사용 시 괄호로 묶기. (연산자 우선순위 혼동 방지.)
> 6. 함수 식별자는 밑줄로 구분해서 표기하기. (ex. test_func)
> 7. 함수에 필수 주석 처리하기.  
>
>
> ### 2. 설계
> 프로그램 소스 코드에 대한 설계는 '설계' 파일에 정리함.
